# Launch plan — 90 days

## Phase 1: MVP (weeks 1-2)
- Vehicle-first data model with 4 categories, ~20 seed vehicles
- AI news pipeline ingesting from 5-10 RSS sources
- Homepage with vehicle category picker
- Vehicle detail page (filtered fix/upgrade/trail/news)
- Article detail pages for all 4 types
- Dark-mode editorial design
- Deploy to Vercel

## Phase 2: Content + community (weeks 3-4)
- Auth (Google + GitHub OAuth)
- Comments on articles
- User story submissions
- Expand vehicle database to 50+ vehicles
- 10+ hand-written articles (fix, upgrade, trail)
- SEO: sitemap, OG tags, RSS output feed

## Phase 3: Discovery (weeks 5-8)
- Trails map view with interactive pins
- Site-wide search across articles and vehicles
- "More like this" recommendations per vehicle
- Newsletter signup (Buttondown or ConvertKit)
- Analytics (Plausible)

## Phase 4: Engagement (weeks 9-12)
- Rig profiles — user-submitted builds with parts lists
- Rig profile linked to articles ("I did this upgrade")
- Vehicle comparison tool
- Content partnerships with off-road publications
- SEO content push: 50+ articles targeting long-tail keywords
